我在後才發現foodomo有網頁的型態，因此把主題改為foodomo校園升級版(主要為新增以下功能)

延伸亮點一：無痛揪團與 AA 制分帳系統 
痛點： 學生或辦公室訂便當、訂飲料，最討厭的就是傳閱菜單、算錢算到頭痛，最後還有人忘記給錢。

UX 延伸設計：

揪團房間： 首頁設計一個明顯的「發起揪團」按鈕。點擊後生成一個專屬網址。

即時協作購物車： 購物車面板上會顯示每個人的頭像（例如：小明點了排骨飯、小華點了珍奶），大家加了什麼料一目了然。

一鍵 AA 結帳： 結帳畫面自動幫你算出每個人要付多少錢，甚至可以透過 LINE Pay 或 icash Pay 直接發送收款連結。

視覺效果： 左邊是菜單，右邊是一個「黏性（Sticky）」的揪團面板，隨著大家點餐，面板會有頭像彈跳出現的微動畫。
