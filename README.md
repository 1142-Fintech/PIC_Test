## 1. 專案簡介

這是一個校園美食外送訂餐平台，主要特色是支援**個人點餐**和**揪團點餐**兩種模式。

- 使用者可以瀏覽多個餐廳的商品，加入購物車後個人結帳
- 或是建立揪團邀請朋友，即時看到所有成員的選餐，最後統一結帳並確認轉帳

---

## 2. 頁面說明

### 2.1 首頁

`src/pages/index.tsx` ・ 路由：`/`

**功能：**

- 展示所有餐廳的商品，分類顯示（星巴克、COLDSTONE、Mister Donut、21世紀、Semeur聖娜）
- 左側欄位為分類篩選按鈕，點選後只顯示該分類商品
- 右側為商品卡片格線，支援加入購物車
- 使用 `getStaticProps` 在建置時預先載入所有商品資料（靜態生成）

**UI 結構：**

```
[左側分類列] | [右側商品格線（2～4欄，依螢幕寬度自適應）]
```

---

### 2.2 結帳頁

`src/pages/checkout.tsx` ・ 路由：`/checkout` 或 `/checkout?groupId={id}`

此頁面根據是否有 `groupId` 參數分為兩種模式。

#### 個人結帳模式（無 groupId）

- 顯示目前購物車內的商品與總金額
- 可切換是否使用點數折抵（固定 320 點）
- 按下確認付款後清空購物車並返回首頁
- 訂單記錄寫入 Firebase `userOrders/{uid}`

#### 揪團結帳模式（有 groupId）

- 即時訂閱 Firebase 揪團資料（`subscribeToGroup`）
- 顯示所有成員的選餐與個人小計
- 每位成員有顏色頭像、名稱、付款狀態
- **非發起人：** 顯示「確認我已轉帳」按鈕，按下後更新付款狀態
- **發起人：** 顯示所有成員付款進度條，以及「確認全員到齊，結束揪團！」按鈕
- 揪團結束後自動將訂單歷史寫入所有成員的 `userOrders/{uid}`
- 揪團關閉後自動導回首頁

> 未登入者會被導向登入頁。

---

### 2.3 登入頁

`src/pages/user/login.tsx` ・ 路由：`/user/login`

**功能：**

- 電子郵件與密碼登入
- 失敗時顯示對應錯誤訊息（帳號不存在、密碼錯誤、嘗試次數過多等）
- 登入成功後依 `?redirect=` 參數導向目標頁，否則回首頁
- 提供連結到註冊頁

---

### 2.4 註冊頁

`src/pages/user/register.tsx` ・ 路由：`/user/register`

**功能：**

- 電子郵件、密碼、確認密碼三欄位
- 前端驗證：密碼需至少 6 字元、兩次輸入需一致
- 後端錯誤處理：密碼強度不足、信箱已被使用等
- 提供連結到登入頁

---

### 2.5 訂單歷史頁

`src/pages/user/history.tsx` ・ 路由：`/user/history`

**功能：**

- 受保護路由，需登入（未登入導向登入頁）
- 即時訂閱 `userOrders/{uid}` 顯示所有歷史訂單
- 依時間由新到舊排列
- 每筆訂單顯示：揪團名稱、日期時間、個人選餐明細、個人小計、整團總計

---

### 2.6 揪團點餐頁

`src/pages/group/[groupId].tsx` ・ 路由：`/group/{groupId}`

**功能：**

- 受保護路由，未登入者導向 `/user/login?redirect=/group/{groupId}`
- 進入後自動將使用者加入揪團（若尚未加入）
- 即時訂閱揪團資料，顯示成員人數、所有成員選餐
- 商品瀏覽介面與首頁相同（左側分類、右側格線）
- 使用者新增或調整商品時，即時更新 Firebase
- 發起人可複製邀請連結分享給朋友
- 右下角浮動購物車顯示所有成員選餐，並提供「進入結帳」按鈕

---

### 2.7 404 頁面

`src/pages/404.tsx` ・ 路由：任何不存在的路徑

顯示簡單的 404 錯誤訊息。

## 3. 元件說明

### 3.1 導覽列

`src/components/Header/`

- **`index.tsx`** — 外層容器，固定在頁面頂端（z-index: 100，高度 4rem）
- **`Content.tsx`** — 導覽列內容

| 區域           | 內容                                   |
| -------------- | -------------------------------------- |
| 左側           | Logo「Foodomo 校園強化版」，點擊回首頁 |
| 右側（已登入） | 使用者頭像下拉選單（歷史訂單、登出）   |
| 右側（未登入） | 「註冊」和「登入」按鈕                 |
| 右側尾端       | 深色／淺色模式切換按鈕                 |

---

### 3.2 商品卡片

`src/components/common/ProductCard.tsx`

每張商品卡片包含：

- 商品圖片（滑入放大 105%）
- 右上角橘色價格標籤
- 商品名稱
- 「加入購物車」按鈕（橘色）

按下按鈕後會觸發 `ADD_ITEM` action 將商品加入購物車。

---

### 3.3 商品列表

`src/components/common/ProductsList.tsx`

接收商品陣列，以格線方式渲染多個 `ProductCard`，顯示分類標題與商品數量。

---

### 3.4 浮動購物車

`src/components/common/GroupCartWidget.tsx`

固定在畫面右下角（z-50），根據所在頁面有兩種模式：

#### 個人模式（首頁）

- 圖示：購物車，顯示商品件數徽章
- 展開側欄後顯示：
  - 我的選餐清單（可調整數量）
  - 小計金額
  - 「建立揪團 & 複製連結」按鈕
  - 「個人結帳」按鈕

#### 揪團模式（揪團點餐頁）

- 圖示：🍱，顯示成員人數徽章
- 展開側欄後顯示：
  - 揪團標題與發起人
  - 所有成員清單（顏色頭像 + 個人小計）
  - 整團總金額
  - 「進入結帳」按鈕

> 結帳頁不顯示此元件。

---

### 3.5 表單欄位

`src/components/common/ChakraField.tsx`

可重用的表單欄位元件，整合 Formik 驗證：

- 支援文字輸入（`input`）和多行文字（`textarea`）
- 驗證通過時顯示綠色 ✓
- 顯示錯誤訊息與提示文字

---

### 3.6 頁尾

`src/components/layout/Footer.tsx`

簡單頁尾，顯示「Powered by 統一資訊」。

## 4. 使用者流程

### 個人點餐流程

```
首頁瀏覽商品
  → 點擊「加入購物車」
  → 點擊浮動購物車「個人結帳」
  → （未登入）→ 導向登入頁 → 登入後回結帳頁
  → 結帳頁確認選餐與金額
  → 選擇是否使用點數折抵
  → 確認付款 → 訂單儲存 → 返回首頁
```

### 揪團點餐流程

```
發起人：
  首頁 → 點擊浮動購物車「建立揪團 & 複製連結」
  → 揪團建立，連結複製至剪貼簿
  → 分享連結給朋友

成員加入：
  點擊連結 → /group/{groupId}
  → （未登入）→ 導向登入頁 → 登入後自動回揪團頁
  → 自動加入揪團
  → 瀏覽商品並選餐（即時同步至 Firebase）

結帳：
  發起人點擊「進入結帳」→ /checkout?groupId={id}
  → 顯示所有成員選餐
  → 非發起人按「確認我已轉帳」
  → 進度條更新
  → 發起人按「確認全員到齊，結束揪團！」
  → 訂單歷史寫入所有成員
  → 揪團關閉 → 自動導回首頁
```

## 5. 常見維護任務

### 新增商品

1. 在 `src/data/products.ts` 對應分類中加入新商品物件（`id` 需唯一）
2. 將商品圖片（命名為 `{id}.jpg`）放至 `public/images/`
3. 在 `src/data/image-paths.ts` 的 `imagePaths` 物件中加入 `{id}: "/images/{id}.jpg"`

### 新增餐廳分類

1. 在 `src/data/products.ts` 新增一個新的 key-value（分類名稱 → 商品陣列）
2. 首頁和揪團頁會自動讀取新分類並顯示對應按鈕

```
│   │   ├── register.tsx  # 註冊頁
│   │   └── history.tsx   # 訂單歷史頁
│   └── group/
│       └── [groupId].tsx # 揪團點餐頁
│
├── components/
│   ├── Header/
│   │   ├── index.tsx     # 導覽列容器
│   │   └── Content.tsx   # 導覽列內容（Logo、使用者選單）
│   ├── common/
│   │   ├── ProductCard.tsx       # 商品卡片

│   └── layout/
│       └── Footer.tsx    # 頁尾
│
├── hooks/
│   ├── auth.tsx          # 身份驗證 Context
│   └── cart/
│       ├── cart.tsx      # 購物車 Context + Reducer
│       └── cart-functions.ts  # 購物車操作函式

├── data/
│   ├── products.ts       # 所有商品資料
│   └── image-paths.ts    # 商品圖片路徑對應

├── utils/

    ├── firebaseClient.ts  # Firebase 前端初始化

public/
└── images/               # 商品圖片（{id}.jpg）
```
